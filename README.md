# 小诗仙 - 儿童AI古诗陪伴应用

一个专为6-12岁儿童设计的AI古诗学习陪伴系统，通过智能对话、诗词推荐、学习记录等功能，让孩子在轻松愉快的氛围中学习中华经典诗词。

## 项目简介

小诗仙是一个结合大语言模型技术和儿童教育理念的智能诗词学习平台。系统以"小诗仙"这一亲切的AI角色与孩子进行互动，通过对话、游戏、故事等多种形式，激发孩子对古诗词的兴趣。

## 核心亮点

- **智能对话**：基于大语言模型，理解孩子的提问，用儿童友好的语言解释诗词
- **个性推荐**：根据孩子的年龄、兴趣、学习进度推荐合适的诗词（唐诗、宋词、元曲）
- **场景感知**：根据天气、时间、季节推荐应景的诗词
- **学习记录**：记录学习进度，形成个性化的学习档案
- **多终端支持**：Web端、小程序端、智能硬件终端全覆盖

## 硬件终端规划

打造一款专为儿童设计的古诗词学习智能硬件：

- **语音交互**：语音唤醒（"小诗仙"），全语音对话，无需屏幕
- **离线缓存**：常用诗词本地存储，弱网可用
- **家长控制**：APP端管理学习内容和使用时长
- **学习报告**：定期推送学习进度和成果

硬件配置：树莓派 + 麦克风阵列 + 扬声器，卡通造型设计

## 技术栈

- **后端**：FastAPI + SQLAlchemy + MySQL
- **AI模型**：阿里云百炼（通义千问）
- **部署**：Docker + Docker Compose

## 快速开始

```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑 .env 填写数据库和API密钥

# 2. 启动服务（Docker）
docker-compose up -d

# 3. 初始化数据
python scripts/init_business_tables.py
python scripts/init_agent_data.py
```
